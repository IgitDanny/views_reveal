
Views Reveal
===============

The Views Reveal module is a Views Style Plugin that can be used to output Views fields in a Reveal.js presentation.

Requirements
============
## Module Dependencies:

Views 3 https://www.drupal.org/project/views

Libraries 2 https://www.drupal.org/project/libraries

## Reveal.js Library:



Description
===========

This module will create a View format that will display nodes as a Reveal.js
presentation.

Reveal settings are available through Format : Settings.

# Installation

1. Download Reveal.js & copy to sites/all/libraries/reveal.

2. Install & enable the Views Reveal module.

## Create a Reveal Presentation

1. Create a View of Fields.

2. Select the "Reveal" Format type and adjust settings to your liking.

3. Include the View as a Page or in a display panel.
